# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added

- **关联计算功能**：支持参数之间的自动计算
  - 百分比型：根据两个数字字段自动计算百分比（如在线率 = 在线数 / 总数 × 100%）
  - 时间型：根据两个日期时间字段计算时长（支持完整日期时间格式，如 2026/06/24 07:11:40）
  - 自定义小数位数：支持0-3位小数
  - 时间单位选择：小时/天（小时单位自动转换为天+小时格式，如25.5小时=1天1.5小时）
  - 支持在修改参数字段时添加关联计算
- **默认值设置**：为参数字段设置固定默认值（如设备总数=20）
  - 默认值自动填充到表单中
  - 支持文本、数字、百分比类型
- **未完成任务显示修改参数入口**：即使任务未完成也能修改参数字段定义
- **参数字段可编辑**：点击已有字段即可修改配置（名称、类型、默认值）
- **导出报告优化**：新增当日概览区块，包含任务统计，适合微信工作群分享
- **编辑已完成参数值**：在首页支持修改已完成任务的参数值
- **历史记录页面创建周期任务**：在历史记录页面直接创建周期任务
  - 支持设置任务名称、描述信息
  - 支持选择执行频率（每天/每周/每月）
  - 支持配置参数字段

- **参数名称重复校验**：添加参数时自动检测名称重复，弹出居中提示
- **Toast 通知组件**：替换原生 `alert()`，居中显示深色圆角提示框，3秒后自动消失

### Fixed

- **周期任务编辑**：修复周期任务无法编辑详情字段的问题
- **计算字段显示**：修复导出报告中计算字段值为0时显示为"-"的问题
- **百分比输入**：修复百分比字段只能输入整数的问题，支持小数输入
- **单位显示**：修复导出报告中时间型计算字段缺少单位的问题
- **修改参数逻辑**：区分"修改参数字段"和"修改已完成参数值"两个功能

### Updated

- **组件架构**：新增 `Toast.tsx` 通知组件、`toastStore.ts` 状态管理
- **类型定义**：扩展 `ParamField` 接口，添加 `defaultValue` 字段
- **README文档**：更新文档以反映新功能
- **CHANGELOG文档**：添加详细的变更记录
- **README文档**：添加历史记录页面创建周期任务功能说明
- **CHANGELOG文档**：添加历史记录页面创建周期任务功能记录

## [1.0.0] - 2026-06-11

### Added

- **任务管理**：添加、编辑、删除任务功能
- **周期任务**：支持临时任务、每日、每周、每月重复任务
- **自定义参数**：支持文本、数字、百分比类型的自定义参数字段
- **任务模板**：创建、编辑、删除模板，从模板快速创建任务
- **导出功能**：导出已完成任务为Markdown文档
- **历史记录**：本周完成统计、每日完成柱状图、周期任务管理
- **数据持久化**：自动保存到 LocalStorage
- **响应式设计**：适配不同屏幕尺寸
