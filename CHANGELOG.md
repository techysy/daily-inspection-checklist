# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added

- **参数字段排序**：支持上移/下移调整参数字段顺序
- **日期时间类型**：新增 datetime 参数类型，支持配置"当前时间"自动填充
  - 勾选"当前时间"复选框后，完成任务时自动填充 `YYYY/MM/DD HH:mm:ss` 格式的当前时间
- **计算时长回退字段**：计算时长支持配置回退时间字段
  - 优先使用主结束时间字段，若为空则自动使用回退字段
  - 适用于"离线时间→恢复时间，无恢复时间则用巡检时间"等场景
- **参数字段二次编辑**：已添加的参数字段支持直接编辑，无需删除重新添加
  - 点击编辑按钮可修改字段所有配置
  - 支持编辑参数名称、类型、默认值、计算配置等
- **模板导出**：支持将任务模板导出为 JSON 文件
  - 在历史记录页面"任务模板"标签页点击"导出模板"按钮
  - 导出的文件可分享给其他用户导入使用
- **历史记录导出**：支持将历史统计数据导出为文本文件
  - 包含统计表格和已完成任务详情
- **数据可视化设置**：图表页面新增设置菜单
  - 支持显示/隐藏趋势图表
- **图表设备名称显示**：图表标题直接显示第一个参数值（设备名称）
  - 无需悬停即可快速识别图表对应的设备
- **echarts-for-react**：新增图表库依赖

### Updated

- **参数变更历史记录**：更新已完成任务的参数时，自动保存上一次的参数值
  - 新增 `completionParamsHistory` 字段，记录每次参数更新的时间戳和快照
  - 导出报告仍显示最新状态，历史数据可用于追溯变更过程
- **历史数据导入**：支持从导出的 TXT/MD 巡检报告导入历史数据
  - 自动解析报告中的任务名称、参数键值对、完成时间
  - 导入前预览解析结果（日期、任务数、参数列表）
  - 支持批量导入，自动识别数值型参数
- **数据可视化页面**：新增独立「数据分析」页面，基于 ECharts 实现参数值趋势折线图
  - 自动提取已完成任务中的数值型参数（数字、百分比、计算字段）
  - 按任务名称 + 参数名称分组展示趋势曲线
  - 支持按任务筛选，平滑曲线 + 面积填充
  - 鼠标悬停显示具体数值和设备名称
- **关联计算功能**：支持参数之间的自动计算
  - 百分比型：根据两个数字字段自动计算百分比（如在线率 = 在线数 / 总数 × 100%）
  - 时间型：根据两个日期时间字段计算时长（支持完整日期时间格式，如 2026/06/24 07:11:40）
  - 自定义小数位数：支持0-3位小数
  - 时间单位选择：小时/天（小时单位自动转换为天+小时格式，如25.5小时=1天1.5小时）
  - 支持在修改参数字段时添加关联计算
  - 支持配置回退时间字段
- **默认值设置**：为参数字段设置固定默认值（如设备总数=20）
  - 默认值自动填充到表单中
  - 支持文本、数字、百分比类型
- **未完成任务显示修改参数入口**：即使任务未完成也能修改参数字段定义
- **参数字段可编辑**：点击编辑按钮即可修改已有字段配置（名称、类型、默认值）
- **导出报告优化**：新增当日概览区块，包含任务统计，适合微信工作群分享
- **编辑已完成参数值**：在首页支持修改已完成任务的参数值
- **历史记录页面创建周期任务**：在历史记录页面直接创建周期任务
  - 支持设置任务名称、描述信息
  - 支持选择执行频率（每天/每周/每月）
  - 支持配置参数字段
- **参数名称重复校验**：添加参数时自动检测名称重复，弹出居中提示
- **Toast 通知组件**：替换原生 `alert()`，居中显示深色圆角提示框，3秒后自动消失

### Fixed

- **echarts-for-react**：修复图表库未安装导致的导入错误

### Fixed

- **周期任务编辑**：修复周期任务无法编辑详情字段的问题
- **计算字段显示**：修复导出报告中计算字段值为0时显示为"-"的问题
- **百分比输入**：修复百分比字段只能输入整数的问题，支持小数输入
- **单位显示**：修复导出报告中时间型计算字段缺少单位的问题
- **修改参数逻辑**：区分"修改参数字段"和"修改已完成参数值"两个功能

- **工具函数**：新增 `importParser.ts` 报告解析工具
- **导航栏**：Header 新增「数据分析」入口
- **依赖更新**：新增 `echarts` + `echarts-for-react` 图表库
- **类型定义**：新增 `ParamsHistoryEntry` 接口，扩展 `Task` 接口添加 `completionParamsHistory` 字段
- **README文档**：更新文档以反映新功能
- **CHANGELOG文档**：添加详细的变更记录

- **CHANGELOG文档**：添加历史记录页面创建周期任务功能记录

## [1.0.0] - 2026-06-11

### Added

- **任务管理**：添加、编辑、删除任务功能
- **周期任务**：支持临时任务、每日、每周、每月重复任务
- **自定义参数**：支持文本、数字、百分比类型的自定义参数字段
- **任务模板**：创建、编辑、删除模板，从模板快速创建任务
- **导出功能**：导出已完成任务为Markdown文档
- **历史记录**：本周完成统计、每日完成柱状图、周期任务管理
- **数据持久化**：自动保存到 LocalStorage
- **响应式设计**：适配不同屏幕尺寸
